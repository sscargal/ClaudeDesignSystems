Notion Design System
+
💡
This design system faithfully reproduces Notion's content-first visual language. Every token, every component, and every interaction pattern reflects real Notion UI decisions.
+
Overview
+
Notion's design philosophy centers on one idea: the content is the product. The UI chrome is intentionally invisible — soft neutrals, barely-there borders, and restrained motion ensure that words, images, and data occupy all the visual attention.
+
Design Principles
+
Content-first — every pixel serves the words, not the shell.
+
Invisible chrome — the interface recedes when you're not interacting with it.
+
Gentle motion — 120ms transitions, never flashy, always purposeful.
+
Warm neutrals — #F7F6F3 sidebar, not stark white or cool gray.
+
Implementation Status
+

+

+
Key Insights
+
The best interface is no interface. Notion's design team internalized this so completely that the product itself became the proof of concept.
+
▶ Color philosophy — why Notion avoids pure white
Pure white creates harsh contrast. Notion's sidebar is #F7F6F3 — a barely-warm off-white that feels like aged paper. It reduces eye strain during long writing sessions and creates a subtle visual hierarchy between the navigation layer and the content canvas.
+
CSS 
:root {
  /* Notion's three surface layers */
  --color-bg:             #FFFFFF;  /* content canvas */
  --color-surface:        #F7F6F3;  /* sidebar & panels */
  --color-surface-hover:  #EFEFEC;  /* interactive hover */

  /* The barely-there border system */
  --color-border-subtle:  #E3E2E0;
  --color-border-default: #D9D9D7;
}

Motion & Animation

Notion's motion language is defined by restraint. Four durations (80–300ms), one gentle easing curve, and six keyframe animations — all purposeful, none decorative.

Duration Scale
⚡
Instant — 80ms
Hover states, tooltip appear, checkbox toggle
--duration-instant
🏃
Fast — 120ms
Button click feedback, drag handle appear, menu item hover
--duration-fast
🚶
Normal — 200ms
Menu open/close, modal appear, sidebar item active
--duration-normal
🐢
Slow — 300ms
Peek panel slide-in, sidebar collapse, toggle block expand
--duration-slow

♿
Accessibility: All animations respect prefers-reduced-motion: reduce. When this system preference is set, animation duration collapses to 0.01ms and iteration counts are forced to 1 — the transition effects are instant but state changes still occur.
