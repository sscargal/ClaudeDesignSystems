Buttons

Overview

Material Design 3 defines 5 button variants ordered by visual emphasis. All buttons use the full pill shape (border-radius: 9999px), label-large typography (14px, weight 500), 40px height, and state layers via ::before pseudo-element.

--btn-height: 40px --shape-button: 9999px --typescale-label-large-size: 14px --state-hover-opacity: 0.08 --state-pressed-opacity: 0.12

All Variants

Each variant shown in default, hover-visible, and disabled states. Hover over buttons to see the 8% state layer.

Text Button

Lowest emphasis. No background or border. Reduced horizontal padding (12px vs 24px). Best for low-stakes actions like "Skip" or inline text links. padding: 0 12px.

Floating Action Button (FAB)

The most prominent button. Used for the primary action of a screen. Three sizes and an extended variant. Background defaults to --color-tertiary-container with shadow-elevation-3.
