Site Navigation (.site-nav) — Sticky header with glassmorphism

The site nav shown above this content is the live example. It is sticky, uses backdrop blur/vibrancy, includes a dark mode toggle, and collapses to a hamburger on mobile.

.site-nav — position: sticky, backdrop-filter: blur(16px) saturate(180%), height: 64px
.site-nav-inner — flex row, max-width: 1200px, auto margins
.brand-logo — SVG mark + wordmark + tagline
.site-nav-links — flex row of nav links with pill hover states
.site-nav-link.active — accent color + accent-light background
.site-nav-actions — theme toggle + CTA button
.nav-hamburger — visible only below 768px, toggles .open on .site-nav-links
