Secondary Button

Warm surface with border. Supporting actions alongside a primary — "Cancel", "Settings", "Learn more".

Ghost Button

No background, label-secondary text. For low-emphasis actions like "Dismiss", toolbar controls, or inline triggers.

Danger Button

Error red fill. Reserved for destructive, irreversible actions — delete, remove, revoke. Always confirm with a dialog first.

Icon Buttons

Square/circular. Use when the action is obvious from context and a label would be redundant. Always include aria-label.

Send Button

Circular coral button. The primary conversation action. Transitions to a disabled state (grey, no-shadow) when the input is empty.
