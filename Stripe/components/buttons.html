Buttons

Stable

Stripe's button system communicates action hierarchy through fill, border, and weight. Primary actions use Blurple fill; secondary uses white with border; ghost uses transparent. All transitions run at 200ms standard easing.

Usage 
<button class="btn btn-primary">
  Get started
</button>

<a href="#" class="btn btn-secondary btn-lg">
  Learn more
</a>
Variants
.btn-primary Primary action
.btn-secondary Secondary action
.btn-ghost Tertiary / minimal
.btn-danger Destructive action
.btn-link Inline text action
