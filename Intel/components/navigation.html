Accessibility

  • Utility nav uses aria-label="Utility navigation"
  • Main nav uses aria-label="Main navigation"
  • Mega menu triggers use aria-haspopup="true" and aria-expanded
  • Logo link uses aria-label="Intel Home" — SVG marked aria-hidden="true"
  • All interactive items are keyboard navigable with visible focus indicators
  • Active state indicated with .active class and blue underline
