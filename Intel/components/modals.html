Modal Anatomy

  • .modal-overlay — Fixed inset, rgba(0,0,0,0.5) scrim, flex center
  • .modal — White bg, border-radius: 0 (sharp Intel), border-top: 3px solid #0071C5, shadow-xl
  • .modal-header — Title + close button, bottom border
  • .modal-body — Main content area with padding
  • .modal-footer — Action buttons, light gray bg, top border
  • Animation — scaleIn (250ms) + backdrop fadeIn
  • Max width — 560px default, can extend to 680px for spec/data views

Accessibility

  • Modal root element uses role="dialog" and aria-modal="true"
  • Title uses aria-labelledby pointing to .modal-title
  • Opening modal moves focus to first focusable element (Close button or first input)
  • Tab key focus is trapped within the modal while open
  • Escape key closes the modal
  • Clicking the overlay (scrim) closes the modal
  • On close, focus returns to the triggering element
