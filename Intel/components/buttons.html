Accessibility

  • All buttons have :focus-visible ring — 2px Intel Blue outline, 2px offset
  • Disabled state uses opacity: 0.38 + pointer-events: none
  • Icon-only buttons require aria-label
  • Minimum 44×44px touch target for all size variants
  • Loading state hides text visually, spinner uses aria-live if paired with status region
  • prefers-reduced-motion: transitions reduced to 0.01ms
