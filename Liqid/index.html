PCIe Fabric Technology

One fabric. Every resource.

Liqid Matrix creates a universal PCIe fabric that connects every physical resource — CPU, GPU, NVMe, and network — into a single composable pool. Any resource can be dynamically assigned to any host in seconds.

  • ⚡
    Sub-microsecond composition
    Compose new resource configurations in under 60 seconds from a software API call.
  • 🔒
    Hardware-isolated tenancy
    PCIe access control lists enforce strict hardware isolation between composed workloads.

Traditional servers waste 30–50% of GPU capacity due to rigid 1:1 host-GPU binding. Liqid's composable fabric pools every GPU across your cluster, assigning them to whichever workload needs them most.
